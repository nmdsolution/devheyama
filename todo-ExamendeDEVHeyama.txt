ExamendeDEVHeyama This is official exam resource document to join the Heyama Dev team General You will build a small system consisting of: A mobile app built with React Native + Expo A web app built with Next.js and shadcn/ui A REST API built with NestJS A database using MongoDB Image upload handled through any S3 bucket service (except AMAZON All applications must communicate with the API. ) If you cannot complete a section, skip it and focus on what you can deliver. Partial solutions are acceptable. 24h You have 7 days MAX. I you finish earlier, thats great, you can directly come to us for review. Project Details: Create an app that manages a collection of “Objects.” An Object has: title (string) description (string) imageUrl (string — stored in S3) createdAt (date) API: Create a simple REST API: 1. POST /objects Accepts title , description , and an uploaded image fileUpload the image to S3 Saves the record in MongoDB 2. GET /objects Returns the list of Objects 3. GET /objects/:id Returns a single Object 4. DELETE /objects/:id Removes the entry from MongoDB Image deletion from S3 Mobile + Web App: Here you will build a mobile and web app that basically do the same things. Create a minimal UI with the ability to: 1. Create an Object Input fields for title & description Pick an image from the device Upload it through the API When an object is created, the change should reflect in realtime on all screens (SocketIo). This means if I add and object on the mobile app, I should see it appear on the web app in realtime 2. List Objects Fetch and display the list from the API Show images using the S3 URLs returned by the API 3. View a Single Object Very basic screen showing the an object details General notes: UI design does not matter. Just make sure its functional and coherent. No deployment is needed. We will check your code on your PC directly. If you feel like you can improve the project by adding features you want, go for it.
