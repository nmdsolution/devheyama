# Decisions & lessons learned

Persistent, cross-run memory for this orchestrator. Read this before starting
a `/dev-task` run; append one entry after finishing (or after hitting a
notable gotcha), so the same mistake or discovery doesn't have to be
rediscovered next time.

One entry per task. Keep each entry short — this is working memory for
another agent to skim, not documentation.

Format:

```
## <task ref or short title> — <date>
Decision/gotcha: <one or two sentences>
Why it mattered: <one sentence>
```
